#SigmaCode - Modern Front-End Learning Platform

Welcome to **SigmaCode**! An interactive web-based platform specifically designed as a media outlet to master front-end web development technologies, ranging from HTML5 and CSS3 to the fundamentals of JavaScript. This project is fully built with a *Mobile-First Design* approach and features *Dynamic Universal Theming* (Light/Dark Mode).

---

##Key Features

* Structured Learning Modules**: High-quality, in-depth interactive articles covering CSS Variables, CSS Selectors, CSS Media Queries, and CSS Margin.
* Asymmetric & Responsive Layout Systems**: Uses real *Box Model* visualizations and *Color Palette cards* to make understanding CSS properties effortless.
* Live Interactive Demo**: Equipped with a screen *viewport simulator* (Mobile, Tablet, Desktop) and dynamic range sliders powered by JavaScript DOM manipulation.
* Universal Dark/Light Mode Switcher**: Features seamless theme-switching toggles synchronized across all web pages utilizing `localStorage` encapsulation.
* Ultra Responsive**: Fully optimized for various screen footprints using *CSS Grid*, *Flexbox Layout*, and a custom *Media Queries Grid System*.

---

##Project Structure & Module Documentation

The platform hosts an intuitive multi-page navigation across several front-end core components, including:
1.  **Mastering CSS Variables**: A comprehensive guide on utilizing *Custom Properties* (`:root`) for runtime-adjustable theme switching.
2.  **CSS Selectors Guide**: Deep dive explanations covering *Universal Selectors*, *Combinators*, up to calculating *Specificity* weight rules.
3.  **Responsive Web Design**: Practical implementation of *Breakpoints templates*, *Mobile-First Approaches*, and optimizing web *Accessibility* (`prefers-reduced-motion`).
4.  **CSS Margin Deep-Dive**: Exploration of external spacing controls, the *Margin Collapse* phenomenon, and advanced engineering with *Negative Margins*.
5.  **Introduction to JavaScript**: Fundamentals of web interactivity, modifying document structures via the DOM, and simulating dynamic backend mock data tracking.

---

##Technology Stack

*   **Core Structure**: HTML5 (Semantic Elements) & JavaScript ES6+ (DOM Manipulation & State Management).
*   **Styling & Layout**: Vanilla CSS3 Custom Properties, CSS Grid, Flexbox, and TailwindCSS (utilized on the portfolio module).
*   **Icons & Typography**: FontAwesome v6.4.0, Boxicons v2.1.4, and the Segoe UI Font System.
